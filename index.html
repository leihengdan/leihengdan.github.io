发表于 2017-12-24

怎样从本地删除git远程仓库里面的文件

    1. git pull
      将远程代码pull到本地，保持本地仓库跟远端仓库同步。

    1. git re 文件名
      然后我们使用git rm命令删掉要删除的文件。

    1. git commit -m "原因"
      删除之后，本地目录下已经删除的文件夹就不再存在了，为了能将远程仓库的相应文件也删除掉，我们要把这次操作提交到上去

    1. git push
      可以看到代码中显示我们删掉了README文件，然后将更改push到远程
